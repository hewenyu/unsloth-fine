# 基本环境

```bash
# 安装unsloth
pip install unsloth

# 升级unsloth
pip install --upgrade unsloth --no-cache-dir --no-deps git+https://github.com/unsloth/unsloth.git

# 安装一些小工具

# bitsandbytes: 一个用于量化和压缩的库
pip install bitsandbytes

# unsloth_zoo: 一个用于unsloth的工具库
pip install unsloth_zoo
```

```bash
# 查看unsloth帮助
unsloth --help
```
